## 常用命令

| 命令 | 说明 |
|------|------|
| `hexo new "title"` | 新建文章 |
| `hexo new page "name"` | 新建页面 |
| `hexo generate` | 生成静态文件 |
| `hexo server` | 启动本地服务器 |
| `hexo deploy` | 部署 |
| `hexo clean` | 清除缓存 |
| `hexo version` | 查看版本 |
| `hexo help` | 查看帮助 |

## 最常用的组合命令

```bash
hexo clean && hexo g && gulp && hexo deploy
```
